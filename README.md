# Tourism-seasonality-detector
A data-driven project to detect seasonal tourism opportunity across Spanish destinations.

**Project Objective**
Tourism in Spain is strongly seasonal. Winter can favor mountain and ski destinations, summer usually strengthens coastal destinations, and major events such as Feria de Abril in Seville can create temporary peaks in demand.

This project uses accommodation data to detect these monthly opportunity patterns. It does not model each event or seasonal driver individually, but it helps identify when and where tourism revenue potential becomes strongest.

The core question is: which Spanish destination maximizes tourism opportunity in each month of the year?

The tool is intended for tourism planning, revenue management, seasonal investment screening and destination benchmarking, showing how tourism opportunity rotates across Spain throughout the calendar year.
